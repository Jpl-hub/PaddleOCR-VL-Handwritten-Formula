# Submission Checklist

## GitHub Repository

- [ ] Repository is public.
- [ ] README includes task, setup, data preparation, training, evaluation, and demo.
- [ ] Training scripts run with documented commands.
- [ ] Evaluation script reports normalized edit distance, exact match, and token F1.
- [ ] Dataset construction report is complete.
- [ ] Annotation guideline is complete.
- [ ] Demo is available locally and, if possible, as a hosted Space.
- [ ] No large data or model artifacts are committed to Git.

## Evaluation Set Package

- [ ] Images/documents are included.
- [ ] Annotation JSONL is included.
- [ ] Task description is included.
- [ ] Evaluation script is included or linked.
- [ ] Dataset report includes source, size, distribution, difficulty, and QC.
- [ ] Consent/provenance records are retained for authenticity verification.
- [ ] Synthetic or augmented data ratio is explicitly reported.

## Hugging Face Model

- [ ] Fine-tuned model files are uploaded.
- [ ] Model card includes base model, data, training settings, metrics, limitations, and citation.
- [ ] Inference example is included.
- [ ] License and dataset constraints are stated.
